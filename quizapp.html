quiz app

This is a quiz app. In which there are questions related to different programming. It also has some levels. This app is especially for college students. This app is a work in progress. This app I am single-handed.There are also dark and light themes that change depending on the phone system. Many variants of the app are still under consideration. This app uses the concept of API.

Main page

Main page which contains profiles and different types of programming quiz topics.

Setting page

The Settings page provides various options like editing profiles, updating account details, deleting accounts, and resetting everything.

Level page

There are different types of levels on the level page and you can solve the questions of that level by solving them.

Quiz page

Quiz page where you have to select options and skip button where you can also skip. It has 60 seconds for the user to answer a question. If the team completes, the automatic question skips to the next question . It also shows the total number of queries in it.

Result page

The result page shows the user the number of correct, incorrect, and skipped questions. And there also shows marks and stars based on that. And there are also buttons like Home, Share, Answer, and Next.

Answer page

The answer page shows all the details of the answer you attempted and the correct answer to all the questions.
